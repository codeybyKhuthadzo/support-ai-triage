A lightweight, logic-driven classification engine designed to automate the prioritization of incoming support tickets.

##  Key Features
- **Sentiment Analysis:** Detects "Urgent" vs. "Standard" language to set priority levels.
- **Smart Routing:** Automatically assigns tickets to Finance, Engineering, or Success departments based on keyword detection.
- **Priority Scoring:** Calculates estimated response times based on the urgency of the issue.

##  Tech Stack
- **Frontend:** Tailwind CSS (Modern, responsive UI)
- **Logic:** Vanilla JavaScript (High-performance string processing)
- **Deployment:** GitHub Pages

##  Business Use Case
In a high-volume Fintech environment, manually sorting tickets is slow. This tool demonstrates how a simple "triage engine" can reduce response times for critical billing and login issues by up to 80%.
